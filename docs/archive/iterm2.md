# Configuration for iTerm2

```shell
# Set font for iTerm2
/usr/libexec/PlistBuddy -c "Set :'New Bookmarks':0:'Normal Font' 'HackNerdFontCompleteM-Regular 18'" ~/Library/Preferences/com.googlecode.iterm2.plist
/usr/libexec/PlistBuddy -c "Set :'New Bookmarks':0:'Use Non-ASCII Font' 'false'" ~/Library/Preferences/com.googlecode.iterm2.plist
/usr/libexec/PlistBuddy -c "Set :'New Bookmarks':0:'ASCII Anti Aliased' 'true'" ~/Library/Preferences/com.googlecode.iterm2.plist
/usr/libexec/PlistBuddy -c "Set :'HideTab' 'false'" ~/Library/Preferences/com.googlecode.iterm2.plist
```

> **Note:** Restart the app for changes to take affect.
